# Label

A styled `<label>` element used by `FormControl`.

## Import
```js
import { Label } from '@frontile/forms';
```

## Usage
```gts preview
import { Label } from '@frontile/forms';

<template>
  <Label for='name'>Name</Label>
</template>
```
