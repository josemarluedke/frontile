# Radio

`Radio` represents a single option within a `RadioGroup`. Use it to
select one value from a set.

## Import 

```js
import { Radio } from '@frontile/forms';
```

## Usage

```gts preview
import { Radio } from '@frontile/forms';

<template>
  <Radio @label='Option' @value='option' />
</template>
```
