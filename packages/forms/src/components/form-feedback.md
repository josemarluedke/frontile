# FormFeedback

Displays validation messages for a form control.

## Import
```js
import { FormFeedback } from '@frontile/forms';
```

## Usage
```gts preview
import { FormFeedback } from '@frontile/forms';

<template>
  <FormFeedback @messages='This field is required' />
</template>
```

## API

<Signature @package="forms" @component="FormFeedback" />
