# Textarea

`Textarea` works similarly to `Input` but renders a multiline `<textarea>`
element.

## Import 

```js
import { Textarea } from '@frontile/forms';
```

## Usage

```gts preview
import { Textarea } from '@frontile/forms';

<template>
  <Textarea @label='Bio' />
</template>
```
