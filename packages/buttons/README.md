# @frontile/buttons

Component Library for Ember Octane apps: Buttons

## Compatibility

- Ember.js v3.16 or above
- Ember CLI v2.13 or above
- Node.js v10 or above

## Installation

```sh
ember install @frontile/buttons
```

### Styles

```js
// tailwind.config.js

module.exports = {
  theme: {
    // ...
  },
  plugins: [require('@frontile/buttons/tailwind')]
};
```
